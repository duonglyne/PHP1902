Các câu lệnh artisan trong laravel
        -  để di chuyển trong thư mục thì dùng câu lệnh cd
        vd cd tên_thư_mục
        còn muốn quay ngược ra ngoài thì cd ..

        - artisan là công cụ các câu lệnh cho phép tạo nhanh các chức năng trong Laravel
        như là : tạo controller, tạo model, tạo seeder, tạo migrate, tạo lại app key bảo mật ...
        - để chạy được lệnh artisan trong Laravel thì cần truy cập vào thư mục gốc của ứng dụng(thư mục mẹ trực tiếp chứa file .env)
        hoặc (thư mục mẹ trực tiếp chứa file artisan.php)

        -  câu lệnh đầu tiên:
        1. php artisan: câu lệnh này sẽ hiển thị tất cả cá câu lệnh artisan kèm theo ý nghĩa của nó

        2. tạo ra một controller:
        php artisan make:controller TestController
        (lưu ý nếu chỉ gõ trên controller thì controller đó sẽ tạo trực tiếp trong folder
        app/Http/Controller)

        3. trong 1 hệ thống phức tạp thì controller sẽ được chia vào nhiều folder khác nhau
        nếu muốn tạo một controller nằm trong một thư mục đường dẫn folder thì gõ như sau
        php artisan make:controller Backend/ProductController
        php artisan make:controller Frontend/ProductController

        4. tạo model trong Laravel
        Model trong Laravel mặc định nằm trực tiếp trong app
        Model mặc định user.php trong folder app
        - tạo ra một folder là "Models" và một model trực tiếp trong nó
        => php artisan make:model Models/TestModel

        - tạo một model nằm trong thư mục cấp 2 của Models

         php artisan make:model Models/Backend/ProductModel
         php artisan make:model Models/Fronteng/ProductModel

        5. tạo migration (file dùng để tạo table trong CSDL mà không cần tương tác với navicat hoặc phpmyadmin)

        php artisan make:migration create_tên_bảng_table

        => php artisan make:migration create_product_table
        sau khi chạy xong câu lệnh trên thì vào folder database/migrations
        sẽ thấy 1 file php có định dạng " thời_gian_tên_migrantion" vừa tao

        php artisan make:migration create_category_table

        6. sau khi migration và cấu hình đúng tên CSDL đã tồn tại trong .env
        => tạo từ migration thành bảng CSDL thật thì chạy : php artisan migrate

        7. tạo auth đơn giản trong laravel

        php artisan make:auth

        8. sau khi tạo một migration
        để đồng bộ CSDL => php artisan migrate
