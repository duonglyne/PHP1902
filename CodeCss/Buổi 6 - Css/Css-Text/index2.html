Text indent ( thụt đầu dòng )

text-indent ( Khoảng cách thụt vào đầu dòng của đoạn văn )

letter-spacing ( khoảng cách giữa các kĩ tự trong đoạn văn )

word-spacing ( Khoản cách giữa các từ trong đoạn văn )

line-height ( Khoảng cách giữa các dòng trong đoạn văn )
