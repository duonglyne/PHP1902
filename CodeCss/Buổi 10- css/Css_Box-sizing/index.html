Css Box-sizing

box-sizing có giá trị mặc định là content-bõ

Content-box : thuộc tính width và height trong css là gias trị của phần content

border-box : thuộc tính width và height trong css là giá trị của cả box-model

Cần phân biệt : width và height của phần content width và height của phần padding width và height của phần boder
