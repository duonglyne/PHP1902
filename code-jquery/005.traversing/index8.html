    Các phương thức filter tổng hợp
    1, first(): lấy filter đầu tiên
    2, last(): lấy filter cuối cùng
    3, eq(chỉ số): lấy ra filter có chỉ số tương ứng
    4, filter(bộ lọc): filter theo bộ lọc
    4, not(bộ lọc): phủ định lại bộ lọc
