1. yêu cầu tạo ra một giao diện có 4 cột trên màn hình Desktop và có 2 cột trên máy tính bảng và có 1 cột trên màn hình điện thoai
  2. Lưới BS có 12 cột => giao diện có 4 cột : 12/4=3 (col-sm-3, col-md-3, col-3)
  3. Lưới BS có 12 cột => giao diện có 3 cột : 12/3=4 (col-sm-4, col-md-4, col-4)
  4. Lưới BS có 12 cột => giao diện có 2 cột : 12/2=6 (col-sm-6, col-md-6, col-6)
  5. Lưới BS có 12 cột => giao diện có 1 cột : 12/1=12 (col-sm-12, col-md-12, col-12)
  1. Giao diện mobile: class col-* (col-1, col-2,...., col-12)
  2. Giao diện tablet : class col-sm-* (col-sm-1, col-sm-2,...., col-sm-12)
  3. Giao diện laptop : class col-md-* (col-md-1, col-md-2,...., col-md-12)
  4. Giao diện PC lớn : class col-lg-* (col-lg-1, col-lg-2,...., col-lg-12)
  5. Giao diện siêu lớn : class col-xl-* (col-xl-1, col-xl-2,...., col-xl-12)
